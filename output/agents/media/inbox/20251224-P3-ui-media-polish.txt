## Task 3: Empty State Illustrations

Create simple illustrations for empty/error states:

### 3.1 No Projects Illustration
- Used on `/projects` page when user has no saved projects
- Concept: Empty folder with plus sign, or rocket waiting to launch
- Size: 200x150px SVG or PNG
- Colors: Slate-200/300 with primary blue accent

### 3.2 Error State Illustration
- Used when API calls fail or something goes wrong
- Concept: Broken chain, warning triangle, or "oops" visual
- Size: 200x150px SVG or PNG
- Colors: Slate-200/300 with amber/red accent

### 3.3 Success/Complete Illustration
- Used when project export is successful
- Concept: Checkmark celebration, rocket launched, or trophy
- Size: 200x150px SVG or PNG

## Design Guidelines

1. **Style**: Clean, modern, slightly rounded edges
2. **Colors**: Use the palette from `5DS_UI_SPECIFICATION.md`
   - Primary: #0052FF
   - Success: #10B981
   - Warning: #F59E0B
   - Error: #EF4444
   - Neutrals: Slate scale
3. **Format**: SVG preferred (scalable, small file size)
4. **Naming**: Lowercase, hyphenated (e.g., `core-features.svg`)
