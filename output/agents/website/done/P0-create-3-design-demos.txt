## Demo 1: Light Hero (White/Cream)

**File**: `output/shared/design/demos/demo-1-light-hero.html`

### Design Spec
- **Hero Background**: White (#FFFFFF) with subtle orange radial glow
- **Hero Text**: Dark stone (#1C1917)
- **Hero Accent**: Orange (#F97316)
- **Content Sections**: Alternate white ↔ stone-50
- **Cards**: White with shadow + border (elevated)
- **Terminal**: Dark slate, elevated with shadow
- **Footer**: Navy (#1E3A5F)

### Color Palette
```css
:root {
  --hero-bg: #FFFFFF;
  --hero-text: #1C1917;
  --section-bg-1: #FFFFFF;
  --section-bg-2: #FAFAF9;
  --accent: #F97316;
  --accent-hover: #EA580C;
  --footer-bg: #1E3A5F;
}
```

### Key Characteristics
- Clean, open, professional
- Like HubSpot, Notion, Airtable
- Orange pops on white background
- Maximum readability
- Terminal stands out as dark element

## Demo 2: Navy Hero (Matches Footer)

**File**: `output/shared/design/demos/demo-2-navy-hero.html`

### Design Spec
- **Hero Background**: Navy (#1E3A5F) - same as footer
- **Hero Text**: White (#FFFFFF)
- **Hero Accent**: Orange (#F97316) - high contrast on navy
- **Transition**: Navy hero fades to white content
- **Content Sections**: White and stone-50
- **Cards**: White with shadow
- **Terminal**: Slate-800, elevated
- **Footer**: Navy (#1E3A5F) - matches hero

### Color Palette
```css
:root {
  --hero-bg: #1E3A5F;
  --hero-text: #FFFFFF;
  --section-bg-1: #FFFFFF;
  --section-bg-2: #FAFAF9;
  --accent: #F97316;
  --accent-light: #FB923C; /* Brighter orange on dark */
  --footer-bg: #1E3A5F;
}
```

### Key Characteristics
- Bookended design (navy top and bottom)
- Premium, corporate feel
- Like Stripe, Linear (dark hero → light content)
- Orange is extremely vibrant on navy
- Creates visual "frame" around content
- Gradient transition from navy to white

### Hero Gradient Transition
```css
.hero::after {
  content: '';
  position: absolute;
  bottom: 0;
  left: 0;
  right: 0;
  height: 150px;
  background: linear-gradient(to bottom, transparent, #FFFFFF);
}
```

## Demo 3: Warm Sand/Cream Hero (Earthy Premium)

**File**: `output/shared/design/demos/demo-3-warm-sand.html`

### Design Spec
- **Hero Background**: Warm cream (#FDF8F3) with soft amber glow
- **Hero Text**: Warm brown (#44403C) - stone-700
- **Hero Accent**: Deep orange (#EA580C) - orange-600
- **Content Sections**: Cream (#FFFBF7) and white
- **Cards**: White with warm shadow
- **Terminal**: Warm slate, amber accents
- **Footer**: Deep brown (#292524) - stone-800

### Color Palette
```css
:root {
  --hero-bg: #FDF8F3;         /* Warm cream */
  --hero-text: #44403C;        /* Stone-700 */
  --hero-accent-glow: rgba(251, 146, 60, 0.1); /* Amber glow */
  --section-bg-1: #FFFBF7;     /* Warmer white */
  --section-bg-2: #FDF8F3;     /* Cream */
  --accent: #EA580C;           /* Deep orange */
  --accent-hover: #C2410C;     /* Orange-700 */
  --footer-bg: #292524;        /* Stone-800 */
  --card-shadow: 0 4px 20px rgba(120, 113, 108, 0.08); /* Warm shadow */
}
```

### Key Characteristics
- Warm, inviting, premium
- Like luxury/lifestyle brands
- Softer than pure white
- Feels handcrafted and personal
- Orange feels more natural/earthy
- Appeals to creative/indie developers

## Technical Requirements

### Each HTML File Must:
1. Be **standalone** (no external dependencies except Google Fonts)
2. Include **complete inline CSS** in `<style>` tags
3. Be **responsive** (works on mobile)
4. Include a **theme toggle button** (fixed bottom-right) if applicable
5. Use **Inter font** from Google Fonts
6. Be **~600-800 lines** of clean, readable code
