## Problem Statement

The website has good individual components but lacks systematic application. The main issues are:

1. **Dark/light mixing** - Random dark sections break the visual flow
2. **Multiple card styles** - 3+ different card designs across sections
3. **No hero transition** - Abrupt darkâ†’light change is jarring
4. **Terminal inconsistencies** - Different styles in each section
5. **4+ background tones** - Should be 2 maximum

### Task 3: Create Unified Card Component

**File**: `website/app/components/ui/Card.tsx` (create or update)

**ONE card style for all uses**:

```tsx
// Card.tsx
import { cn } from "@/lib/utils";

interface CardProps {
  children: React.ReactNode;
  className?: string;
  hover?: boolean;
}

export function Card({ children, className, hover = true }: CardProps) {
  return (
    <div 
      className={cn(
        // Base styles - NEVER CHANGE THESE
        "bg-white rounded-xl border border-stone-200 p-6",
        "shadow-sm",
        // Hover effect (optional)
        hover && [
          "transition-all duration-200",
          "hover:border-orange-200",
          "hover:shadow-md",
          "hover:shadow-orange-500/5",
          "hover:-translate-y-1"
        ],
        className
      )}
    >
      {children}
    </div>
  );
}

// Card Header with icon
export function CardHeader({ 
  icon, 
  title, 
  status 
}: { 
  icon: React.ReactNode; 
  title?: string;
  status?: React.ReactNode;
}) {
  return (
    <div className="flex items-start justify-between mb-4">
      <div className="w-12 h-12 rounded-lg bg-orange-50 text-orange-500 flex items-center justify-center text-xl">
        {icon}
      </div>
      {status}
    </div>
  );
}
```
