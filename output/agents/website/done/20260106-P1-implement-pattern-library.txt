## Patterns to Implement

### 1. HeroCenteredGradient (Priority: P0)

```
┌────────────────────────────────────────────┐
│           [Optional Badge/Tag]              │
│                                             │
│       Large Headline Goes Here              │
│                                             │
│     Supporting text that explains the       │
│        value proposition clearly            │
│                                             │
│    [Primary CTA]     [Secondary CTA]        │
│                                             │
│    ░░░░░ Gradient Background ░░░░░          │
└────────────────────────────────────────────┘
```

**Props**:
- `headline: string`
- `subheadline: string`
- `badge?: { text: string; variant: 'new' | 'beta' | 'custom' }`
- `primaryCTA: { text: string; href: string }`
- `secondaryCTA?: { text: string; href: string }`
- `variant: 'dark' | 'light' | 'gradient'`

**Inspired by**: linear.app, vercel.com

## Requirements

1. **Use existing design system** - Use `globals.css` colors, Tailwind classes
2. **TypeScript types** - Full props interface for each component
3. **Responsive** - Mobile-first, works on all viewports
4. **Accessible** - Proper ARIA labels, keyboard navigation
5. **Themeable** - Support dark/light variants via props or CSS variables
6. **Animations** - Subtle entry animations using CSS or Framer Motion
