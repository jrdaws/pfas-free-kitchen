## Implementation Steps

### Step 1: Update Tailwind Config

File: `website/tailwind.config.js`

Add/update theme colors:

```javascript
theme: {
  extend: {
    colors: {
      primary: {
        50: '#FFF7ED',
        100: '#FFEDD5',
        200: '#FED7AA',
        300: '#FDBA74',
        400: '#FB923C',
        500: '#F97316',
        600: '#EA580C',
        700: '#C2410C',
        800: '#9A3412',
        900: '#7C2D12',
      },
    },
  },
}
```

### Step 2: Global Search & Replace

In `website/` directory, replace class names:

```
bg-indigo-* → bg-orange-*
text-indigo-* → text-orange-*
border-indigo-* → border-orange-*
ring-indigo-* → ring-orange-*
focus:ring-indigo-* → focus:ring-orange-*
hover:bg-indigo-* → hover:bg-orange-*
hover:text-indigo-* → hover:text-orange-*

bg-gray-* → bg-stone-*
text-gray-* → text-stone-*
border-gray-* → border-stone-*
divide-gray-* → divide-stone-*
```

### Step 3: Add Premium Button Shadows

Update button components with enhanced shadows:

```tsx
// Before
className="bg-orange-500 hover:bg-orange-600 ..."

// After  
className="bg-orange-500 hover:bg-orange-600 shadow-md hover:shadow-lg transition-all duration-200 ..."
```

### Step 4: Dark Mode Verification

Test these pages in both light and dark modes:
- [ ] Homepage (`/`)
- [ ] Configurator (`/configurator`)
- [ ] All configurator steps
- [ ] Preview panels
- [ ] Export flow

### Step 5: Update CSS Variables (if used)

Check `globals.css` or similar for any CSS custom properties using old colors.
