# Task: Create Feature Code Templates
Priority: P1
Agent: Website Agent
Date: 2026-01-03

## Objective
Create code templates for the 3 most commonly selected features. These templates get included in generated projects when users select these features.

## Feature Templates to Build

### 1. Shopping Cart Feature
Location: `packages/templates/features/ecommerce/shopping-cart/`

Required files:
- `files.json` - Manifest with dependencies
- `lib/cart/cart-context.tsx` - React context for cart state
- `lib/cart/cart-utils.ts` - Price formatting, totals, etc.
- `components/CartDrawer.tsx` - Slide-out cart panel
- `components/CartItem.tsx` - Individual cart item display
- `components/AddToCartButton.tsx` - Add to cart trigger
- `hooks/useCart.ts` - Cart operations hook

Features:
- Add/remove/update quantities
- Persistent storage (localStorage)
- Cart totals calculation
- Empty cart state

### 2. User Registration Feature  
Location: `packages/templates/features/user-management/email-registration/`

Required files:
- `files.json` - Manifest (depends on auth integration)
- `components/SignupForm.tsx` - Email/password registration
- `components/LoginForm.tsx` - Login form
- `components/ForgotPasswordForm.tsx` - Password reset request
- `components/ResetPasswordForm.tsx` - Set new password
- `lib/auth/validation.ts` - Form validation helpers
- `hooks/useAuth.ts` - Auth state and actions hook

Features:
- Email/password validation
- Password strength indicator
- Remember me option
- Redirect after auth

### 3. Admin Dashboard Feature
Location: `packages/templates/features/enterprise/admin-dashboard/`

Required files:
- `files.json` - Manifest
- `app/(admin)/layout.tsx` - Admin layout with sidebar
- `app/(admin)/dashboard/page.tsx` - Overview page
- `app/(admin)/users/page.tsx` - User management
- `components/admin/AdminSidebar.tsx` - Navigation sidebar
- `components/admin/StatsCard.tsx` - Metric display card
- `components/admin/DataTable.tsx` - Sortable data table
- `lib/admin/permissions.ts` - Role-based access helpers

Features:
- Protected routes (admin only)
- Stats overview cards
- User CRUD operations
- Role/permission checks
