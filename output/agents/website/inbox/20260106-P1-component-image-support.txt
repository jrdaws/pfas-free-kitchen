## Implementation Pattern

For each component:

1. Add image props to interface:
```typescript
interface ComponentProps {
  // ... existing
  image?: string;
  backgroundImage?: string;
  previewMode?: boolean;
}
```

2. Render image when available:
```typescript
{image && (
  <div 
    className="w-full h-48 bg-cover bg-center rounded-lg"
    style={{ backgroundImage: `url(${image})` }}
  />
)}
```

3. Or use img tag with error handling:
```typescript
{image && (
  <img 
    src={image}
    alt=""
    className="w-full h-48 object-cover rounded-lg"
    onError={(e) => {
      // Hide broken image or show placeholder
      e.currentTarget.style.display = 'none';
    }}
  />
)}
```
