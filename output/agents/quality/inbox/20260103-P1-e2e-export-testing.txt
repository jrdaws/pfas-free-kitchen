## Test Process

1. **Configure project** via /configure page
2. **Download ZIP** via export
3. **Extract and install**:
   ```bash
   unzip project.zip -d test-project
   cd test-project
   npm install
   ```
4. **Build project**:
   ```bash
   npm run build
   ```
5. **Verify structure**:
   - Check .dd/ folder exists
   - Check all integration folders present
   - Verify component imports resolve

## Validation Checklist
