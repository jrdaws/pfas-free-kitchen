Design and implement the database schema for the My Projects dashboard.

1. Create Supabase migration for `projects` table:

   - template (text)

   - created_at, updated_at
   
2. Create RLS policies for user ownership
3. Create API route: POST /api/projects/create
4. Create API route: GET /api/projects/list
