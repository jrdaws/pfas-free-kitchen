## Deployment Steps

1. **Verify Build**

   ```bash
   cd /Users/joseph.dawson/Documents/dawson-does-framework/website
   npm run build
   ```

2. **Deploy to Vercel**
   ```bash
   # User must authenticate first
   vercel login
   
   # Then deploy
   vercel --prod
   ```

3. **Configure Environment Variables**
   In Vercel dashboard, add:
   - `NEXT_PUBLIC_SUPABASE_URL`
   - `NEXT_PUBLIC_SUPABASE_ANON_KEY`
   - `SUPABASE_SERVICE_ROLE_KEY`
   - `STABILITY_API_KEY` (optional, for image generation)

4. **Verify Deployment**
   - Check all pages load
   - Test configurator flow
   - Verify export works
