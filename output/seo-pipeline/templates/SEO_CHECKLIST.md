# SEO Content Checklist

> **Use this checklist for every piece of SEO content**
> **Minimum passing score: 80/100**

---

## On-Page SEO (40 points)

### Title & Meta
- [ ] **Title tag** includes primary keyword (5 pts)
- [ ] **Title** is 50-60 characters (3 pts)
- [ ] **Meta description** includes keyword (3 pts)
- [ ] **Meta description** is 150-160 chars with CTA (2 pts)

### URL & Structure
- [ ] **URL** is short and keyword-rich (3 pts)
- [ ] **H1** matches search intent (3 pts)
- [ ] **Only one H1** on page (2 pts)

### Keywords
- [ ] **Primary keyword** in first 100 words (3 pts)
- [ ] **Primary keyword** in at least one H2 (2 pts)
- [ ] **Secondary keywords** in H2s/H3s (3 pts)
- [ ] **Keyword density** 1-2% (not stuffed) (2 pts)

### Links
- [ ] **3-5 internal links** to relevant pages (4 pts)
- [ ] **2-3 external links** to authoritative sources (3 pts)
- [ ] **Descriptive anchor text** (not "click here") (2 pts)

**On-Page Score: ___ / 40**

## Content Quality (40 points)

### Depth & Length
- [ ] **Word count** meets/exceeds top competitors (5 pts)
- [ ] **Comprehensive coverage** of topic (5 pts)
- [ ] **Answers main search question** clearly (5 pts)

### Uniqueness
- [ ] **Original insights** not found in competitors (5 pts)
- [ ] **Specific examples** (not generic) (3 pts)
- [ ] **Current information** (2025 data/stats) (2 pts)

### Readability
- [ ] **Flesch score 60+** (8th-grade level) (3 pts)
- [ ] **Short paragraphs** (2-4 sentences) (2 pts)
- [ ] **Bullet points/lists** for scanability (2 pts)
- [ ] **Visual breaks** every 300-500 words (2 pts)

### AI Quality
- [ ] **Varied sentence structure** (not repetitive) (3 pts)
- [ ] **Specific numbers** (not "many" or "several") (2 pts)
- [ ] **Would pass as human-written** (1 pt)

**Content Quality Score: ___ / 40**

---

## Technical SEO (20 points)

### Images
- [ ] **All images have alt text** (3 pts)
- [ ] **Alt text is descriptive** (includes keyword where natural) (2 pts)
- [ ] **Images compressed** (WebP format, < 200KB) (2 pts)

### Structured Data
- [ ] **Article schema** implemented (3 pts)
- [ ] **FAQ schema** if FAQ section exists (2 pts)
- [ ] **Schema validates** (no errors) (2 pts)

### Mobile & Speed
- [ ] **Mobile-friendly** layout (2 pts)
- [ ] **Page loads < 3s** (2 pts)

### Other
- [ ] **Canonical URL** set correctly (1 pt)
- [ ] **Page in sitemap** (1 pt)

**Technical Score: ___ / 20**

## Quick Fixes for Common Issues

| Issue | Fix |
|-------|-----|
| Title too long | Remove brand name or use shorter synonym |
| No keyword in first 100 words | Add natural mention in intro |
| Missing internal links | Link to related blog posts/pages |
| Generic examples | Replace with specific case studies |
| Walls of text | Add headers, bullets, images |
| AI-sounding content | Add personal anecdotes, opinions |

## Featured Snippet Optimization

For "how to" queries:
- [ ] Numbered steps (not bullets)
- [ ] Each step starts with action verb
- [ ] Total steps visible without scrolling

For definition queries:
- [ ] 40-60 word definition immediately after H2
- [ ] Starts with "[Term] is..."

For list queries:
- [ ] Clear H2 with keyword
- [ ] Bulleted list immediately following
- [ ] 8-10 items optimal
