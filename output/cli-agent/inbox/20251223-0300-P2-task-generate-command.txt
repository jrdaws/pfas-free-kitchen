## Context

The AI generation engine is complete in `packages/ai-agent`. We now need to expose this functionality through the CLI so users can generate projects locally without using the web interface.

## Task Objectives

1. **Implement `framework generate` Command**
   - Add new command to `bin/framework.js`
   - Accept project description interactively or via flags
   - Call the ai-agent package to generate project

2. **Command Interface**
   ```bash
   framework generate [options]
   
   Options:
     --description, -d  Project description (required)
     --name, -n         Project name
     --template, -t     Template to use (saas, flagship-saas, seo-directory)
     --tier             Model tier (fast, balanced, quality)
     --output, -o       Output directory (default: ./)
     --stream           Show streaming progress (default: true)
   ```

3. **Interactive Mode**
   - If no description provided, prompt user
   - Show progress spinner/bars during generation
   - Display token usage at end

## Success Criteria

- [ ] `framework generate -d "..." --name "project"` works
- [ ] Interactive prompt when no description provided
- [ ] Streaming progress displayed in terminal
- [ ] Token usage summary at end
- [ ] Tests added for new command
