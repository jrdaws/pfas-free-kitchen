## IMPLEMENTATION

### 1. Pass Context to Export

When exporting, include context from configurator:

```javascript
// In export command or pull command
const projectContext = {
  name: config.projectName,
  vision: config.context?.vision || null,
  mission: config.context?.mission || null,
  successCriteria: config.context?.successCriteria || null,
  description: config.inspiration?.description || null,
};
```

### 2. Generate Dynamic START_PROMPT

```javascript
function generateStartPrompt(context) {
  let prompt = `# ${context.name} - Start Prompt\n\n`;
  
  if (context.vision || context.mission || context.successCriteria) {
    prompt += `## Pre-filled Context\n\n`;
    if (context.vision) prompt += `**Vision**: ${context.vision}\n\n`;
    if (context.mission) prompt += `**Mission**: ${context.mission}\n\n`;
    if (context.successCriteria) prompt += `**Success Criteria**: ${context.successCriteria}\n\n`;
    prompt += `---\n\n`;
  }
  
  prompt += `## Additional Context Needed\n\n`;
  prompt += `Answer these before proceeding:\n\n`;
  prompt += `- TARGET USER:\n`;
  prompt += `- MVP SCOPE:\n`;
  prompt += `- TIMELINE:\n`;
  
  return prompt;
}
```

### 3. Add Toggle in Configurator (Optional)

In Context Fields step, add checkbox:
```tsx

  id="includeInStartPrompt"
  checked={includeContext}
  onCheckedChange={setIncludeContext}
/>
<Label htmlFor="includeInStartPrompt">
  Include this context in exported START_PROMPT.md
</Label>
```
