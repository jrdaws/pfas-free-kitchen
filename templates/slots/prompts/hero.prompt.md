# Hero Section Content Generation

Generate compelling hero section content for a web application.

## Context

**Project Name:** {{projectName}}
**Project Type:** {{projectType}}
**Description:** {{projectDescription}}
**Target Audience:** {{targetAudience}}
**Voice/Tone:** {{voiceTone}}

## Requirements

Generate the following content:

### 1. Headline (5-10 words)
- Capture the core value proposition
- Use action-oriented language
- Make it memorable and specific
- Avoid generic phrases like "Welcome to..." or "The best..."

### 2. Subtext (15-25 words)
- Expand on the headline with benefits
- Address the target audience's pain point
- Include a subtle call to action
- Keep it scannable and clear

### 3. Primary CTA Button Text (2-4 words)
- Action-oriented verb + outcome
- Create urgency without being pushy
- Examples: "Start Building", "Get Started Free", "Try for Free"

### 4. Secondary CTA Text (optional, 2-4 words)
- Lower commitment action
- Examples: "Watch Demo", "Learn More", "See Pricing"

## Output Format

```json
{
  "headline": "Your compelling headline here",
  "subtext": "Supporting text that expands on the value proposition and encourages action.",
  "ctaText": "Start Free",
  "secondaryCtaText": "Watch Demo"
}
```

## Style Guidelines

- Professional but approachable
- Focus on benefits, not features
- Use "you" language to address the reader directly
- Match the {{voiceTone}} tone specified
