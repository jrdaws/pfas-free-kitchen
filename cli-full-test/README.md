# SaaS Template

Full-featured SaaS application template with authentication, billing, and user management.

## Features

- User authentication and authorization
- Subscription billing
- User management dashboard
- Multi-tenant architecture

## Getting Started

```bash
npm install
npm run dev
```

Visit http://localhost:3000 to see your app.

## Tech Stack

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
