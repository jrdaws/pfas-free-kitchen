## Design Principles

These prompts enforce:

1. **No AI-generic styling**
   - NO purple gradients
   - NO Inter/Roboto fonts
   - NO generic color schemes

2. **Professional e-commerce patterns**
   - Filter sidebar (Newegg style)
   - Product cards with verification badges
   - Comparison tray (Newegg style)
   - Mega menu navigation (Wayfair style)

3. **Typography**
   - DM Sans for headings
   - Source Sans 3 for body

4. **Color palette**
   - Primary: Deep teal (#0D5C63)
   - Accent: Warm amber (#E8A838)
   - Tier-specific colors for verification badges
