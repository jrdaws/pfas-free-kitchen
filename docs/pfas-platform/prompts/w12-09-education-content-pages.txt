TASK 2: HOW WE VERIFY PAGE
================================================================================

Create `src/pfas-web/app/learn/how-we-verify/page.tsx`:

```
┌───────────────────────────────────────────────────────────────────────────────┐
│ Learn > How We Verify Products                                                │
├───────────────────────────────────────────────────────────────────────────────┤
│                                                                               │
│                   Our Verification Process                                    │
│                   ────────────────────────                                    │
│                                                                               │
│  We use a tiered verification system to rate products based on the           │
│  quality of evidence available. Higher tiers = more confidence.              │
│                                                                               │
├───────────────────────────────────────────────────────────────────────────────┤
│                                                                               │
│  The Four Verification Tiers                                                  │
│  ───────────────────────────                                                  │
│                                                                               │
│  ┌─────────────────────────────────────────────────────────────────────────┐ │
│  │ ✓✓ TIER 4 · MONITORED                                     Highest      │ │
│  │ ────────────────────────────────────────────────────────────────────── │ │
│  │ Products with ongoing testing programs. We receive updated lab        │ │
│  │ results on a schedule (typically annually). If a product fails        │ │
│  │ retest, the listing is immediately updated.                           │ │
│  │                                                                        │ │
│  │ Evidence required:                                                     │ │
│  │ • Initial lab report (Tier 3 equivalent)                              │ │
│  │ • Documented retest schedule                                          │ │
│  │ • Access to ongoing results                                           │ │
│  └─────────────────────────────────────────────────────────────────────────┘ │
│                                                                               │
│  ┌─────────────────────────────────────────────────────────────────────────┐ │
│  │ ✓ TIER 3 · LAB TESTED                                                  │ │
│  │ ────────────────────────────────────────────────────────────────────── │ │
│  │ Products tested by an independent laboratory using validated methods.  │ │
│  │ We verify the lab report is authentic and the testing method is        │ │
│  │ appropriate for detecting PFAS.                                        │ │
│  │                                                                        │ │
│  │ Evidence required:                                                     │ │
│  │ • Lab report from accredited laboratory                               │ │
│  │ • Test method (LC-MS/MS or total fluorine screening)                  │ │
│  │ • Detection limits stated (LOD/LOQ)                                   │ │
│  │ • Sample description matching product                                 │ │
│  └─────────────────────────────────────────────────────────────────────────┘ │
│                                                                               │
│  ┌─────────────────────────────────────────────────────────────────────────┐ │
│  │ ○ TIER 2 · POLICY REVIEWED                                             │ │
│  │ ────────────────────────────────────────────────────────────────────── │ │
│  │ We've reviewed the manufacturer's PFAS policy, ingredient             │ │
│  │ specifications, or supply chain documentation. No independent          │ │
│  │ testing, but credible documentation exists.                           │ │
│  │                                                                        │ │
│  │ Evidence required:                                                     │ │
│  │ • Written PFAS-free policy on company letterhead                      │ │
│  │ • Material safety data sheets (MSDS)                                  │ │
│  │ • Supply chain certifications                                         │ │
│  └─────────────────────────────────────────────────────────────────────────┘ │
│                                                                               │
│  ┌─────────────────────────────────────────────────────────────────────────┐ │
│  │ · TIER 1 · BRAND STATEMENT                                   Lowest    │ │
│  │ ────────────────────────────────────────────────────────────────────── │ │
│  │ The brand has made a public claim that the product is PFAS-free, but  │ │
│  │ we have not verified this claim with documentation or testing.         │ │
│  │                                                                        │ │
│  │ Evidence required:                                                     │ │
│  │ • Public statement on brand website or packaging                      │ │
│  │ • No independent verification                                         │ │
│  └─────────────────────────────────────────────────────────────────────────┘ │
│                                                                               │
├───────────────────────────────────────────────────────────────────────────────┤
│                                                                               │
│  Component-Level Analysis                                                     │
│  ────────────────────────                                                     │
│                                                                               │
│  We don't just verify "the product" - we examine each component that         │
│  contacts food. A pan might have:                                            │
│                                                                               │
│  [Diagram: Pan with labeled parts]                                           │
│  • Cooking surface (food contact) ← Verified                                 │
│  • Rivets (food contact) ← Verified                                          │
│  • Handle (non-food contact) ← May not be verified                           │
│                                                                               │
│  This matters because some products use PFAS-free coatings but have          │
│  PFAS in handles, lids, or other components.                                 │
│                                                                               │
├───────────────────────────────────────────────────────────────────────────────┤
│                                                                               │
│  Our Independence                                                             │
│  ────────────────                                                             │
│                                                                               │
│  • We do not accept payment for reviews or rankings                          │
│  • Affiliate links are clearly disclosed                                     │
│  • Verification tier is never influenced by affiliate relationships          │
│  • All evidence documents are stored with SHA-256 hashes                     │
│                                                                               │
│  [Read Our Editorial Policy]                                                 │
│                                                                               │
└───────────────────────────────────────────────────────────────────────────────┘
```
